Uncertainty

Data analysis depends on udnerlying uncertainties for each measurement.Each measurement made, a sensible estimate of the uncertainty of measurement is needed.
Word "uncertainty" or "error" are interchangeably used. An experimental uncertainty is an attempt to quantify the level to which the value being measured or calculated can reasonably be known.
For a measured value \(x\), its uncertainty \(\Delta x\). These 2 parts form the best estimate of the true value, written as \(x\pm\Delta x\). Also the true value of the quantity, we lable \(X\).
Only one unit is needed and the value should always be rounded to same placeholder level. e.g. \(x=(156\pm2)\mu m\) or \(x=(156\pm2)\times 10^{-6}m\).

Standard error

As mentioned before. An experimental uncertainty is an attempt to quantify the level to which the value being measured or calculated can reasonably be known. We need to understand the word 'reasonable'. By convention, the quoted uncertainty (e.g.\(\Delta x=\pm2\mu m\))represents \(\pm 1\) standard deviation in the experimental results.
This quoted uncertainty is called the Standard Error.
About our experimental value for which we are \(68\%\) confident that \(x-\Delta x\leq X \leq x+\Delta x\) (\(\pm1\) standard error), \(95\%\) confident that \(x-2\Delta x\leq X \leq x+2\Delta x\) (\(\pm2\) standard error) and \(99.7\%\) confident that \(x-3\Delta x\leq X \leq x+3\Delta x\) (\(\pm3\) standard error).
Symbol \(\Delta\) and \(\sigma\) are often used interchangeably to represent standard errors.
